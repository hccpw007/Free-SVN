# 字体来源说明

当前目录需要以下 JetBrains Mono 字体文件用于代码编辑器的等宽字体显示：

## 下载方式

1. 访问 JetBrains Mono 官方 Release 页面：
   https://github.com/JetBrains/JetBrainsMono/releases

2. 下载最新版本的 `JetBrainsMono-x.xxx.zip`

3. 解压后从 `fonts/woff2/` 目录中提取以下文件到当前目录：
   - `JetBrainsMono-Regular.woff2`
   - `JetBrainsMono-Bold.woff2`
   - `JetBrainsMono-Italic.woff2`
   - `JetBrainsMono-BoldItalic.woff2`

## 替代方案

若字体文件缺失，浏览器会自动回退到系统等宽字体（如 Monaco、Consolas），不影响功能使用。
