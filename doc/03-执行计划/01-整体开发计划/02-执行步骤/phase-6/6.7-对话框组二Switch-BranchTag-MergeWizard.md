## 操作细节

### 6.7A — SwitchDialog

el-dialog 460px。当前路径只读 + 目标路径下拉（EL-Select filterable allow-create，prepend `^/branches` 分支列表）+ 手动输入 + ignoreAncestry 复选框。切换中按钮 loading。

**流程：** `svnStore.listBranches()` → 下拉选单 → `svnStore.switchBranch()` → toast + 关闭。

### 6.7B — BranchTagDialog

el-dialog 460px。来源 URL 自动填充（显示当前工作副本 URL）→ 目标路径输入（placeholder 提示 branches/tags 下）→ 日志 textarea → 版本号可选 ElInputNumber → [创建] 按钮。

**流程：** `svnStore.copyBranchTag()` → toast + 关闭。

### 6.7C — MergeWizard（4 步向导）

el-dialog 520px，`close-on-click-modal=false`。步骤进度条在顶部（①→②→③→④，完成后 ✅ 标记）。

| 步骤 | 内容 | 按钮 |
|------|------|------|
| 1: 选择来源 | 源 URL 输入 | [下一步] |
| 2: 版本范围 | 起始/结束版本号（ElInputNumber） | [上一步] [下一步] |
| 3: 选项 | 3 个 checkbox（ignoreAncestry/recordOnly/allowMixed） | [上一步] [下一步] |
| 4: 执行 | 摘要信息 + [执行合并] + 结果面板 | [上一步] [完成] |

**结果三种状态：** `success`→绿色"合并完成，无冲突"；`conflict`→琥珀色"N 个文件存在冲突"；`error`→红色错误信息。
