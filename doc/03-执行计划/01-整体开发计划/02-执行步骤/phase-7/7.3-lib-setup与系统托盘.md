### 7.3A — 修改 `src-tauri/src/lib.rs`

**插件注册（setup 前）：**
- `tauri_plugin_store` — 配置持久化
- `tauri_plugin_single_instance` — 单实例防护
- `tauri_plugin_window_state` — 窗口状态持久化
- `tauri_plugin_log` — 日志系统
- `tauri_plugin_dialog` — 系统对话框

**setup 回调：**
1. `logging::logger::init(app.handle())` — 初始化日志
2. `config::store::init(app.handle())` — 初始化配置存储
3. 系统托盘创建（TrayIconBuilder + MenuBuilder）

**系统托盘菜单：**
```
显示主窗口
──────────
SVN 更新当前工作副本
SVN 清理当前工作副本
──────────
关于 Free-SVN
──────────
退出
```

**托盘事件：** 单击托盘图标→显示/激活主窗口；菜单项→对应操作；退出→`app.exit(0)`。

**单实例回调：** 第二实例启动时获取首实例 WebviewWindow，调用 `.show()` + `.set_focus()`。

**窗口配置（tauri.conf.json）：** `minWidth: 800, minHeight: 500`（视觉设计 §2.2）。推荐尺寸 width: 1200, height: 800。

**关闭窗口→最小化到托盘：**

App.vue 的 onMounted 中监听 `onCloseRequested` 事件，调用 `window.hide()` 阻止退出：

```ts
import { getCurrentWebviewWindow } from '@tauri-apps/api/webviewWindow'

onMounted(() => {
  getCurrentWebviewWindow().onCloseRequested(async () => {
    await getCurrentWebviewWindow().hide()
  })
})
```

**command 注册（generate_handler! 共 25 个）：** 全部 commands 模块的公开函数。
