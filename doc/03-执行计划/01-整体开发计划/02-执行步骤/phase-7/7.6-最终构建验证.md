## 操作细节

### 7.6A — 验证顺序

按顺序执行，逐项修复到零报告：

1. `cd src-tauri && cargo build` — 修复编译错误/警告（unused import/variable、unwrap）
2. `cd src-tauri && cargo test` — 修复测试失败
3. `cd src-tauri && cargo clippy` — 修复 clippy 警告
4. `cnpm run build` — 修复 TS 类型错误和 Vite 错误
5. `cnpm run lint` — 修复 ESLint 错误
6. `cnpm run knip` — 删除未引用文件，重复 4-6 直到零报告
7. 最终命令：`cargo build && cargo test && cnpm run build && cnpm run lint`

### 7.6B — 常见问题

| 问题 | 修复 |
|------|------|
| 未使用变量 | 删除或 `_` 前缀 |
| 未使用 import | 删除 |
| 类型不匹配 | 对齐 Rust/TS 类型定义 |
| generate_handler! 缺失 | 补全注册 |
| knip 未使用文件 | 确认后删除 |
