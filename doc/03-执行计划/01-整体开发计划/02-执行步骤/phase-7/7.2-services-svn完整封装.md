### 7.2A — 新建 `src/services/svn.ts`

前端唯一调用后端入口。28 个 invoke 方法 + 3 个 event listener，统一通过 `wrappedInvoke` 实现 try-catch + 错误码翻译。

**统一调用包装：**
```ts
import { invoke } from '@tauri-apps/api/core'
import { listen } from '@tauri-apps/api/event'
import type { FileItem, RepoInfo, DiffResult, LogEntry, BlameLine, MergeResult, OperationProgress, OperationResult } from '@/types/svn'

/** 统一 invoke 包装：自动 try-catch + 错误码翻译 */
async function wrappedInvoke<T>(cmd: string, args?: Record<string, unknown>): Promise<T> {
  try { return await invoke<T>(cmd, args) }
  catch (e) {
    const msg = typeof e === 'string' ? e : e instanceof Error ? e.message : '未知错误'
    throw translateError(msg)
  }
}

/** 错误码 → 用户可读 i18n key */
function translateError(msg: string): string {
  // 1. 优先匹配后端 AppError 错误码前缀
  if (msg.startsWith('SVN_TIMEOUT')) return 'error.operationTimeout'
  if (msg.startsWith('SVN_CANCELLED')) return 'error.operationCancelled'
  if (msg.startsWith('SVN_NOT_FOUND')) return 'error.svnNotFound'
  if (msg.startsWith('NETWORK_UNREACHABLE')) return 'error.networkUnreachable'
  if (msg.startsWith('SVN_OPERATION_IN_PROGRESS')) return 'error.operationInProgress'
  if (msg.startsWith('SVN_EXEC_FAILED')) return 'error.svnCommandFailed'
  if (msg.startsWith('SVN_AUTH_FAILED')) return 'error.authenticationFailed'
  if (msg.startsWith('INVALID_INPUT')) return 'error.invalidInput'
  if (msg.startsWith('IO_ERROR')) return 'error.ioError'
  if (msg.startsWith('REPO_ERROR')) return 'error.repoError'
  if (msg.startsWith('SVN_PARSE_FAILED')) return 'error.parseFailed'

  // 2. 匹配命令执行错误中的 SVN CLI E- 码
  if (msg.includes('E155037')) return 'error.workingCopyClean'
  if (msg.includes('E155010') || msg.includes('E200009')) return 'error.pathNotFound'
  if (msg.includes('E204899')) return 'error.notWorkingCopy'
  if (msg.includes('E170001') || msg.includes('认证')) return 'error.authenticationFailed'
  if (msg.includes('E160013') || msg.includes('E731001') || msg.includes('404')) return 'error.urlNotFound'
  if (msg.includes('E155035') || msg.includes('locked')) return 'error.workingCopyLocked'
  if (msg.includes('E205007') || msg.includes('E155024')) return 'error.invalidRevision'
  if (msg.includes('E165001') || msg.includes('E195020')) return 'error.conflictDetected'
  if (msg.includes('E200012') || msg.includes('E200014')) return 'error.pathAlreadyVersioned'
  if (msg.includes('E155036')) return 'error.entryNotFound'
  if (msg.includes('timeout') || msg.includes('超时')) return 'error.operationTimeout'
  return 'error.unknown'
}
```

**导入与导出约定：**

`services/svn.ts` 是**前端唯一直接调用 `invoke` 的文件**。`stores/svn.ts`（Pinia store）应从本文件导入方法，不得直接 `import { invoke }`：

```ts
// stores/svn.ts — 正确方式
import { getStatus, createCommit } from '@/services/svn'
```

### 7.2B — 重构 settings store（从直接 invoke 切换为 services/svn.ts）

settings store（`src/stores/settings.ts`）在步骤 4.2 中暂允许直接调用 `invoke`，本步骤完成 services/svn.ts 后，须将其重构：

**修改要点：**
1. 移除 `import { invoke } from '@tauri-apps/api/core'`
2. 添加 `import { loadSettings, saveSettings } from '@/services/svn'`
3. `load()` 方法改为：
   ```ts
   async function load() {
     try {
       const s = await loadSettings()
       defaultCheckoutDir.value = s.defaultCheckoutDir
       globalIgnorePattern.value = s.globalIgnorePattern
       // ... 其余字段赋值
       isDirty.value = false
     } catch { /* 默认值 */ }
   }
   ```
4. `save()` 方法改为：
   ```ts
   async function save() {
     await saveSettings({
       defaultCheckoutDir: defaultCheckoutDir.value,
       globalIgnorePattern: globalIgnorePattern.value,
       diffTool: diffTool.value,
       mergeTool: mergeTool.value,
       showUnversioned: showUnversioned.value,
       language: language.value,
       autoStart: autoStart.value,
       darkMode: darkMode.value,
     })
     isDirty.value = false
   }
   ```
