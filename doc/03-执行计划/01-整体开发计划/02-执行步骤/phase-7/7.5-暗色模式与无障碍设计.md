### 7.5A — App.vue 暗色初始化

```ts
import { useSettingsStore } from '@/stores/settings'
onMounted(async () => {
  const s = useSettingsStore(); await s.load()
  if (s.darkMode === true) document.documentElement.classList.add('dark')
  else if (s.darkMode === false) document.documentElement.classList.remove('dark')
  else { // system
    const mq = window.matchMedia('(prefers-color-scheme: dark)')
    document.documentElement.classList.toggle('dark', mq.matches)
    mq.addEventListener('change', (e) => {
      if (s.darkMode === undefined) document.documentElement.classList.toggle('dark', e.matches)
    })
  }
})
```

- 背景色：`bg-white` → `dark:bg-slate-800`，`bg-slate-50` → `dark:bg-slate-800/50`

### 7.5C — 无障碍清单

| 要求 | 实现方式 | 代码模板 |
|------|---------|---------|
| 4.5:1 对比度 | Tailwind 色板默认满足 | 无需额外处理 |
| 焦点指示器 2px blue-400 | 所有 button/a/input 元素统一追加 `focus:ring-2 focus:ring-blue-400 focus:outline-none` | 见下方逐角色示例 |
| 颜色+图标双通道 | FileStatusIcon 使用图标+文字双通道表示状态，不依赖颜色 | 已实现 |
| aria-label | 所有图标按钮（含 Lucide 图标）补全 | `<button :aria-label="t('toolbar.diff')"><FileDiffIcon /></button>` |
| aria-role | 对话框使用 `role="dialog"`，状态提示使用 `role="alert"` | Element Plus el-dialog 自带 dialog role |
| prefers-reduced-motion | Tailwind v4 motion-safe/motion-reduce 变体 + style.css 全局降级 | 见下方模板 |
| 文本缩放 | 全部使用 Tailwind `text-sm`/`text-xs` 等相对单位 | 无 px 硬编码 |

**焦点指示器逐角色模板（需在对应组件的 template 中应用）：**

```vue
<!-- 按钮（按钮类）-->
<button class="... focus:ring-2 focus:ring-blue-400 focus:outline-none focus:ring-offset-2">...</button>

<!-- 链接类 -->
<a class="... focus:ring-2 focus:ring-blue-400 focus:outline-none">...</a>

<!-- 输入框（el-input 自动处理，仅自定义输入框需要）-->
<input class="... focus:ring-2 focus:ring-blue-400 focus:outline-none focus:border-blue-400" />

<!-- 可点击图标（如密码显示/隐藏）-->
<button @click="toggle"
  class="focus:ring-2 focus:ring-blue-400 focus:outline-none rounded"
  :aria-label="showPassword ? t('auth.hidePassword') : t('auth.showPassword')">
  <component :is="showPassword ? EyeOff : Eye" class="w-4 h-4" />
</button>

<!-- 设置页切换开关 -->
<el-switch class="focus:ring-2 focus:ring-blue-400" :aria-label="t('settings.darkMode')" />
```

- `focus:ring-offset-2` 用于按钮类元素，避免 ring 与按钮边框重叠

**`prefers-reduced-motion` 实现：**

Tailwind CSS v4 内置 `motion-safe:` 和 `motion-reduce:` 变体，无需额外配置。在动画组件（如对话框打开/关闭）中添加：

```ts
// 在 Element Plus 组件上（如需自定义过渡）：
<div class="motion-safe:transition-all motion-safe:duration-200 motion-reduce:transition-none">
      <!-- 或在 app 级别使用 CSS 媒体查询统一禁用：src/style.css -->
  <!-- 动画内容 -->
</div>
```

如需全局禁用所有动画，在 `src/style.css` 中添加：

```css
@media (prefers-reduced-motion: reduce) {
  *, *::before, *::after { animation-duration: 0.01ms !important; animation-iteration-count: 1 !important; transition-duration: 0.01ms !important; }
}
```

### 7.5D — JetBrains Mono 等宽字体加载

DiffViewer 和 LogPage 等代码展示区域使用 JetBrains Mono 等宽字体以优化可读性。

**字体文件打包方案：**

1. 下载 JetBrains Mono 字体文件（TTF 或 WOFF2 格式），放置于 `src-tauri/resources/fonts/JetBrainsMono/` 目录：
   ```
   src-tauri/resources/fonts/JetBrainsMono/
   ├── JetBrainsMono-Regular.woff2
   ├── JetBrainsMono-Bold.woff2
   └── JetBrainsMono-Italic.woff2
   ```

2. 在 `tauri.conf.json` 中配置资源打包，确保字体文件被包含在应用包中：
   ```json
   {
     "bundle": {
       "resources": {
         "resources/fonts/**": "fonts/"
       }
     }
   }
   ```

3. 在 `src/style.css` 或 `index.html` 的 `<style>` 中定义 `@font-face`：
   ```css
   @font-face {
     font-family: 'JetBrains Mono';
     src: url('/fonts/JetBrainsMono/JetBrainsMono-Regular.woff2') format('woff2');
     font-weight: 400;
     font-style: normal;
     font-display: swap;
   }
   @font-face {
     font-family: 'JetBrains Mono';
     src: url('/fonts/JetBrainsMono/JetBrainsMono-Bold.woff2') format('woff2');
     font-weight: 700;
     font-style: normal;
     font-display: swap;
   }
   @font-face {
     font-family: 'JetBrains Mono';
     src: url('/fonts/JetBrainsMono/JetBrainsMono-Italic.woff2') format('woff2');
     font-weight: 400;
     font-style: italic;
     font-display: swap;
   }
   ```

4. 全局应用：在 Tailwind CSS 配置中设置 `font-mono` 族：
   ```ts
   // tailwind.config.ts 或使用 CSS 中的 @theme
   @theme {
     --font-mono: 'JetBrains Mono', ui-monospace, monospace;
   }
   ```

5. 在 DiffViewer.vue 和 LogPage.vue 等代码容器上使用 `font-mono` 类名。

**备选方案（开发阶段）：** 使用系统等宽字体（无需下载字体文件），`font-mono` 默认回退到系统等宽字体，开发阶段无需字体文件即可正常显示。
**

> **Vite 构建说明：** Tauri v2 中的静态资源（包括 `fonts/` 目录）通过 Vite 的 `public/` 目录引用。字体文件在构建时自动复制到输出目录。在组件中引用字体文件时使用相对于 `public/` 的路径，如 `url(/fonts/JetBrainsMono/...)`。
