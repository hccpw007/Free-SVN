| 文件路径 | 操作类型 |

## 操作细节

### 7.5A — App.vue 暗色初始化

```ts
import { useSettingsStore } from '@/stores/settings'
onMounted(async () => {
  const s = useSettingsStore(); await s.load()
  if (s.darkMode === true) document.documentElement.classList.add('dark')
  else if (s.darkMode === false) document.documentElement.classList.remove('dark')
  else { // system
    const mq = window.matchMedia('(prefers-color-scheme: dark)')
    document.documentElement.classList.toggle('dark', mq.matches)
    mq.addEventListener('change', (e) => {
      if (s.darkMode === undefined) document.documentElement.classList.toggle('dark', e.matches)
    })
  }
})
```

### 7.5B — 暗色检查清单

逐组件全文件搜索补全，执行以下命令定位所有需要添加 `dark:` 变体的位置：

```bash
# 搜索每个类名并逐行检查添加上下文
grep -rn 'bg-white' src/ --include='*.vue' --include='*.ts'
grep -rn 'bg-slate-50' src/ --include='*.vue'
grep -rn 'text-slate-900\|text-slate-700\|text-slate-600' src/ --include='*.vue'
grep -rn 'text-slate-500' src/ --include='*.vue'
grep -rn 'border-slate-200\|border-slate-300' src/ --include='*.vue'
grep -rn 'hover:bg-slate-50' src/ --include='*.vue'
# 确认无 shadow 用法的暗色降级
grep -rn 'shadow-' src/ --include='*.vue' | grep -v 'shadow-none'
```

替换指南：

| 亮色 | 暗色 | 搜索目标 |
|------|------|---------|
| bg-white | dark:bg-slate-800 | 卡片/面板/弹窗 |
| bg-slate-50 | dark:bg-slate-800/50 | 次要背景 |
| text-slate-900 | dark:text-slate-50 | 主文字 |
| text-slate-600/700 | dark:text-slate-300 | 正文 |
| text-slate-500 | dark:text-slate-400 | 次要文字 |
| border-slate-200/300 | dark:border-slate-700 | 边框 |
| hover:bg-slate-50 | dark:hover:bg-slate-700/50 | hover 背景（行状态，视觉设计 §3.2）|
| bg-blue-50 | dark:bg-blue-900/20 | 选中行背景 |
| bg-white border | dark:bg-slate-800 dark:border-slate-600 | 输入框 |

暗色特殊处理：shadow-none + 边框替代阴影，Diff 使用 `dark:bg-red-900/30` 和 `dark:bg-green-900/30`。

### 7.5C — 无障碍清单

| 要求 | 实现 |
|------|------|
| 4.5:1 对比度 | Tailwind 色板满足 |
| 焦点指示 | 所有交互元素统一 `focus:ring-2 focus:ring-blue-400 focus:outline-none`。具体在每个 `.vue` 文件中搜索 button/a/input 标签逐一补全，示例：
```vue
<button class="... focus:ring-2 focus:ring-blue-400 focus:outline-none">...</button>
```
使用 `grep -rn '<button\|<a\|<input' src/ --include='*.vue' | grep -v 'focus:'` 定位缺少焦点指示器的元素后逐文件补全 |
| 颜色+图标双通道 | FileStatusIcon 已实现 |
| aria-label | 所有图标按钮补全 |
| prefers-reduced-motion | 见下方 Tailwind v4 配置 |
| 文本缩放 | 全部使用 Tailwind 类（`text-sm`、`text-xs`），无 px 硬编码 |

**`prefers-reduced-motion` 实现：**

Tailwind CSS v4 内置 `motion-safe:` 和 `motion-reduce:` 变体，无需额外配置。在动画组件（如对话框打开/关闭）中添加：

```ts
// 在 Element Plus 组件上（如需自定义过渡）：
<div class="motion-safe:transition-all motion-safe:duration-200 motion-reduce:transition-none">
      <!-- 或在 app 级别使用 CSS 媒体查询统一禁用：src/style.css -->
  <!-- 动画内容 -->
</div>
```

如需全局禁用所有动画，在 `src/style.css` 中添加：

```css
@media (prefers-reduced-motion: reduce) {
  *, *::before, *::after { animation-duration: 0.01ms !important; animation-iteration-count: 1 !important; transition-duration: 0.01ms !important; }
}
```
