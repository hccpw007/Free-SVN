## 操作细节

### 7.5A — App.vue 暗色初始化

```ts
import { useSettingsStore } from '@/stores/settings'
onMounted(async () => {
  const s = useSettingsStore(); await s.load()
  if (s.darkMode === true) document.documentElement.classList.add('dark')
  else if (s.darkMode === false) document.documentElement.classList.remove('dark')
  else { // system
    const mq = window.matchMedia('(prefers-color-scheme: dark)')
    document.documentElement.classList.toggle('dark', mq.matches)
    mq.addEventListener('change', (e) => {
      if (s.darkMode === undefined) document.documentElement.classList.toggle('dark', e.matches)
    })
  }
})
```

### 7.5B — 暗色检查清单

逐组件全文件搜索补全：

| 亮色 | 暗色 | 搜索目标 |
|------|------|---------|
| bg-white | dark:bg-slate-800 | 卡片/面板/弹窗 |
| bg-slate-50 | dark:bg-slate-800/50 | 次要背景 |
| text-slate-900 | dark:text-slate-50 | 主文字 |
| text-slate-600/700 | dark:text-slate-300 | 正文 |
| text-slate-500 | dark:text-slate-400 | 次要文字 |
| border-slate-200/300 | dark:border-slate-700 | 边框 |
| hover:bg-slate-100 | dark:hover:bg-slate-700 | hover 背景 |

暗色特殊处理：shadow-none + 边框替代阴影，Diff 使用 `dark:bg-red-900/30` 和 `dark:bg-green-900/30`。

### 7.5C — 无障碍清单

| 要求 | 实现 |
|------|------|
| 4.5:1 对比度 | Tailwind 色板满足 |
| 焦点指示 | 所有交互元素 `focus:ring-2 focus:ring-blue-400` |
| 颜色+图标双通道 | FileStatusIcon 已实现 |
| aria-label | 所有图标按钮补全 |
| prefers-reduced-motion | `motion-safe:` 前缀 |
