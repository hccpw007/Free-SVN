## 操作细节

全屏 `fixed inset-0 z-50` 半透明遮罩，居中卡片（w-80）。显示线性进度条（el-progress striped）、阶段文字、文件计数（font-mono）、取消按钮。使用 Teleport 到 body。

```vue
<script setup lang="ts">
import { ref, computed } from 'vue'
import { useSvnStore } from '@/stores/svn'
import { useI18n } from 'vue-i18n'

const { t } = useI18n()
const svnStore = useSvnStore()

// i18n key 说明：以下所有文字均通过 t() 使用 i18n key，无硬编码
// 已在语言包中定义的 key：
//   common.cancel          = "取消" / "Cancel"
//
// progress 段需要在 locale 文件中补充定义：
//   progress.operationInProgress = "操作进行中" / "Operation in Progress"
//   progress.processing          = "处理中..." / "Processing..."
//   progress.filesProcessed      = "已处理 {count} 个文件" / "{count} files processed"
//   progress.cancelling          = "正在取消..." / "Cancelling..."
// 以上 key（除 common.cancel 外）在 src/locales/zh-CN.json 和 src/locales/en.json 的 "progress" 段中定义
//
// 取消操作的文字处理：
// - 按钮默认显示 t('common.cancel')（"取消" / "Cancel"），由 handleCancel 触发
// - 点击后显示 t('progress.cancelling')（"正在取消..." / "Cancelling..."），isCancelling=true 控制
// - cancelOperation() 完成后界面关闭（isRunning=false 导致 v-if 隐藏整个遮罩）
// - 操作被取消后 svnStore 会触发 error 回调，其中使用 error.SVN_CANCELLED = "操作已取消" / "Operation cancelled"
const isCancelling = ref(false)
const progress = computed(() => svnStore.progress)
const isRunning = computed(() => svnStore.isOperationRunning)

async function handleCancel() {
  isCancelling.value = true
  try { await svnStore.cancelOperation() }
  finally { isCancelling.value = false }
}
</script>

<template>
  <Teleport to="body">
    <div v-if="isRunning" class="fixed inset-0 z-50 flex items-center justify-center bg-black/40 dark:bg-black/60">
      <div class="w-80 p-6 rounded-lg bg-white dark:bg-slate-800 shadow-lg border border-slate-200 dark:border-slate-700">
        <h3 class="text-base font-semibold text-slate-900 dark:text-slate-50 mb-4">
          {{ progress?.operation || t('progress.operationInProgress') }}
        </h3>
        <el-progress :percentage="progress?.percent ?? 0" :stroke-width="8" striped striped-flow :duration="3" class="mb-3" />
        <p class="text-sm text-slate-500 dark:text-slate-400 mb-1">{{ progress?.stage || t('progress.processing') }}</p>
        <p class="text-xs text-slate-400 dark:text-slate-500 font-mono mb-5">{{ t('progress.filesProcessed', { count: progress?.fileCount ?? 0 }) }}</p>
        <button
          class="w-full py-2 px-4 rounded-md text-sm font-medium transition-colors duration-150 focus:ring-2 focus:ring-blue-400 focus:outline-none"
          :class="isCancelling
            ? 'bg-slate-200 dark:bg-slate-700 text-slate-400 cursor-not-allowed'
            : 'bg-red-50 dark:bg-red-900/20 text-red-600 dark:text-red-400 hover:bg-red-100 dark:hover:bg-red-900/40'"
          :disabled="isCancelling"
          @click="handleCancel"
        >
          {{ isCancelling ? t('progress.cancelling') : t('common.cancel') }}
        </button>
      </div>
    </div>
  </Teleport>
</template>
```

**svnStore 依赖接口：**

```ts
// stores/svn.ts 需要暴露
const isOperationRunning = ref(false)
const progress = ref<OperationProgress | null>(null)
interface OperationProgress {
  operation: string; percent: number; stage: string; fileCount: number
}
async function cancelOperation(): Promise<void>
```

**全局规则：** ProgressOverlay 显示期间，所有工具栏按钮禁用（opacity-50 + tooltip）、文件行操作按钮禁用、键盘仅 Esc 生效。
