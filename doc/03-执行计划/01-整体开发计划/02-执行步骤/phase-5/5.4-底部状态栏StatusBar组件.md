## 操作细节

纯展示组件，高度 28px，`bg-slate-100 dark:bg-slate-800/50`，顶部 1px 边框。三段信息 `justify-between` 分布。

```vue
<script setup lang="ts">
import { computed } from 'vue'
import { useWorkspaceStore } from '@/stores/workspace'
import { useI18n } from 'vue-i18n'

const { t } = useI18n()
const ws = useWorkspaceStore()

// 所有文字全部通过 t() 使用 i18n key，无硬编码
// statusBar.revision = "版本 {rev}" / "Revision {rev}"
// statusBar.revisionNone = "无版本信息" / "No revision info"
// statusBar.lastCommit = "上次提交：{time}" / "Last commit: {time}"
// statusBar.svnVersion = "SVN {ver}" / "SVN {ver}"
// statusBar.copyRevision = "点击复制版本号" / "Click to copy revision"
// 以上 key 在 src/locales/zh-CN.json 和 src/locales/en.json 的 "statusBar" 段中定义

const revision = computed(() => ws.currentRevision ? t('statusBar.revision', { rev: ws.currentRevision }) : t('statusBar.revisionNone'))
const commitTime = computed(() => ws.lastCommitTime ? t('statusBar.lastCommit', { time: ws.lastCommitTime }) : '')
const svnVer = computed(() => ws.svnVersion ? t('statusBar.svnVersion', { ver: ws.svnVersion }) : '')

async function copyRev() {
  if (ws.currentRevision) {
    await navigator.clipboard.writeText(String(ws.currentRevision))
  }
}
</script>

<template>
  <div class="h-7 px-4 flex items-center justify-between bg-slate-100 dark:bg-slate-800/50 border-t border-slate-200 dark:border-slate-700 text-xs">
    <span class="text-slate-500 dark:text-slate-400 font-mono cursor-pointer hover:text-green-600 transition-colors duration-150" :title="t('statusBar.copyRevision')" @click="copyRev">{{ revision }}</span>
    <span v-if="commitTime" class="text-slate-500 dark:text-slate-400 font-mono" :title="commitTime">{{ commitTime }}</span>
    <span v-if="svnVer" class="text-slate-400 dark:text-slate-500 font-mono" :title="svnVer">{{ svnVer }}</span>
  </div>
</template>
```

**交互：** 版本号点击复制 revision 到剪贴板，hover 变绿色。全段 `font-mono text-xs`。

**`lastCommitTime` 更新时机说明：**

`lastCommitTime` 由 `workspaceStore` 管理，在以下 3 个时机触发更新：

| 时机 | 触发位置 | 操作 |
|------|---------|------|
| 进入工作副本时 | `HomePage.vue` 的 `onMounted` | 调用 `getInfo()`获取工作副本信息后设置 `lastCommitTime` |
| 手动刷新文件列表时 | 各页面 refresh 逻辑中 | 调用 `getInfo()` 获取最新提交时间 |
| 提交操作完成后 | `svnStore.commit()` 调用后 | 在 commit 成功回调中主动调用 `getInfo()` 更新 `lastCommitTime` |

**示例代码：在 HomePage.vue onMounted 中更新 lastCommitTime：**

```ts
import { useWorkspaceStore } from '@/stores/workspace'
import { useI18n } from 'vue-i18n'
import { getInfo } from '@/services/svn'

onMounted(async () => {
  if (workspaceStore.currentPath) {
    try {
      const info = await getInfo(workspaceStore.currentPath)
      workspaceStore.lastCommitTime = info.lastCommitDate ?? ''
      workspaceStore.currentRevision = info.revision ?? 0
    } catch { /* 静默处理 */ }
  }
})
```
