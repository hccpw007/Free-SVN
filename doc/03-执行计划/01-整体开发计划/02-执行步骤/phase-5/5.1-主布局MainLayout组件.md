## 操作细节

### 5.1A — 新建 `src/layouts/MainLayout.vue`

使用 `<script setup lang="ts">` 语法，实现 flex-col 四层布局结构，总高度 `h-screen`：

```
┌──────────────────────────────────────────┐
│  TopBar (48px, h-12) 固定               │
├──────────────────────────────────────────┤
│  ToolBar (40px, h-10) 固定               │
├──────────────────────────────────────────┤
│  <RouterView /> (flex-1 min-h-0) 填充    │
├──────────────────────────────────────────┤
│  StatusBar (28px, h-7) 固定              │
└──────────────────────────────────────────┘
```

**组件结构：**

```vue
<script setup lang="ts">
import { computed } from 'vue'
import { useRoute } from 'vue-router'
import { useWorkspaceStore } from '@/stores/workspace'
import TopBar from '@/components/common/TopBar.vue'
import ToolBar from '@/components/common/ToolBar.vue'
import StatusBar from '@/components/common/StatusBar.vue'
import ProgressOverlay from '@/components/svn/ProgressOverlay.vue'

const route = useRoute()
const workspaceStore = useWorkspaceStore()

// 欢迎页模式：route.name === 'HomePage' 且无 currentPath 时隐藏 ToolBar 和 StatusBar
const isWelcomePage = computed(() => {
  return route.name === 'HomePage' && !workspaceStore.currentPath
})
</script>

<template>
  <div class="h-screen flex flex-col bg-slate-50 dark:bg-slate-900">
    <!-- TopBar: 48px 固定高度 -->
    <TopBar class="shrink-0" />

    <!-- ToolBar: 40px 固定高度，欢迎页模式隐藏 -->
    <ToolBar v-if="!isWelcomePage" class="shrink-0" />

    <!-- RouterView: 弹性填充，溢出滚动 -->
    <main class="flex-1 min-h-0 overflow-auto">
      <RouterView />
    </main>

    <!-- StatusBar: 28px 固定高度，欢迎页模式隐藏 -->
    <StatusBar v-if="!isWelcomePage" class="shrink-0" />

    <!-- ProgressOverlay: 全屏遮罩，通过 v-if 控制显示 -->
    <ProgressOverlay />
  </div>
</template>
```

**布局详细说明：**

| 层级 | 高度 | Tailwind 类 | 说明 |
|------|------|-------------|------|
| 外层容器 | 100vh | `h-screen flex flex-col` | 全屏 flex 纵向排列 |
| TopBar | 48px | `h-12 shrink-0` | 顶部状态栏，不缩放 |
| ToolBar | 40px | `h-10 shrink-0` | 工具栏，`v-if` 控制显示 |
| RouterView 容器 | 弹性 | `flex-1 min-h-0 overflow-auto` | 填充剩余空间，内容溢出滚动 |
| StatusBar | 28px | `h-7 shrink-0` | 底部状态栏，`v-if` 控制显示 |
| ProgressOverlay | 全屏 | — | `position: fixed` 全屏遮罩，始终渲染但 `v-if` 控制 |

**欢迎页模式判断逻辑：**

```ts
const isWelcomePage = computed(() => {
  // 条件1: 当前路由是 HomePage（route.name === 'HomePage'）
  // 条件2: workspaceStore.currentPath 为空（无工作副本）
  return route.name === 'HomePage' && !workspaceStore.currentPath
})
```

- `route.name` 来自 Vue Router 配置中路由的 `name` 属性
- `workspaceStore.currentPath` 由 workspace store 管理
- 两个条件同时满足时才视为欢迎页模式
- 欢迎页模式下隐藏 ToolBar 和 StatusBar（TopBar 始终显示）
- 当用户检出或打开工作副本后，`workspaceStore.currentPath` 被设置，自动进入完整布局

**暗色模式适配：**
- 外层容器使用 `bg-slate-50 dark:bg-slate-900`
- 内部子组件各自处理自己的暗色适配
- 不使用 `<style>` 标签或自定义 CSS

**ProgressOverlay 位置：**
- 放在 MainLayout 的 `<template>` 最底部
- ProgressOverlay 组件内部使用 `position: fixed` 全屏覆盖
- 通过 svnStore.isOperationRunning 控制显示/隐藏
- ProgressOverlay 本身不依赖 MainLayout 的布局层级
