### 4.3A — 修改 `src/router/index.ts`

```ts
import { createRouter, createWebHistory } from 'vue-router'
import type { RouteRecordRaw } from 'vue-router'

const routes: RouteRecordRaw[] = [
  {
    path: '/',
    redirect: '/workspace',
  },
  {
    path: '/workspace',
    component: () => import('@/layouts/MainLayout.vue'),
    children: [
      {
        path: '',
        name: 'HomePage',
        component: () => import('@/pages/HomePage.vue'),
      },
      {
        path: 'commit',
        name: 'CommitPage',
        component: () => import('@/pages/CommitPage.vue'),
      },
      {
        path: 'log',
        name: 'LogPage',
        component: () => import('@/pages/LogPage.vue'),
      },
      {
        path: 'diff',
        name: 'DiffViewer',
        component: () => import('@/components/svn/DiffViewer.vue'),
      },
      {
        path: 'resolve',
        name: 'ResolveConflict',
        component: () => import('@/components/dialogs/ResolveConflict.vue'),
      },
    ],
  },
  {
    path: '/settings',
    name: 'SettingsPage',
    component: () => import('@/pages/SettingsPage.vue'),
  },
]

export default createRouter({
  history: createWebHistory(),
  routes,
})
```

**路由表：**

| 路由 | 布局 | 组件 | 说明 |
|------|------|------|------|
| `/` | — | — | 重定向到 /workspace |
| `/workspace` | MainLayout | HomePage | 欢迎页 / 变更列表 |
| `/workspace/commit` | MainLayout | CommitPage | 提交面板 |
| `/workspace/log` | MainLayout | LogPage | 日志视图 |
| `/workspace/diff?file=xxx` | MainLayout | DiffViewer | 差异对比 |
| `/workspace/resolve?file=xxx` | MainLayout | ResolveConflict | 冲突解决 |
| `/settings` | 独立 | SettingsPage | 设置页 |

**语言切换联动逻辑：**

settings store 的 `language` 字段值须与 vue-i18n 的 `locale` 联动。在 `src/main.ts` 中 `createI18n` 创建后，在 `App.vue` 的 `onMounted` 中添加以下逻辑：

```ts
// App.vue — 语言切换联动
import { onMounted } from 'vue'
import { useI18n } from 'vue-i18n'
import { useSettingsStore } from '@/stores/settings'

onMounted(() => {
  const settingsStore = useSettingsStore()
  const { locale } = useI18n()

  // 从 settings 中加载语言偏好
  settingsStore.load().then(() => {
    if (settingsStore.language && settingsStore.language !== 'system') {
      locale.value = settingsStore.language
    }
  })
})
```

此外，设置页保存语言变更后，应立即更新 locale：

```ts
// SettingsPage.vue 保存设置后
const { locale } = useI18n()
async function handleSave() {
  await settingsStore.save()
  if (settingsStore.language !== 'system') {
    locale.value = settingsStore.language
  }
}
```

**i18n key 分组规范：**

| 前缀 | 用途 | 示例 |
|------|------|------|
| `app.` | 应用级文案 | welcomeTitle, description |
| `error.` | 后端错误码翻译 | SVN_TIMEOUT, SVN_AUTH_FAILED |
| `common.` | 通用按钮/提示 | cancel, confirm, save |
| `workspace.` | 工作副本视图 | noChanges, pathNotExist |
| `file.` | 文件状态/操作 | statusModified, actionDiff |
