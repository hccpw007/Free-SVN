## §2 关键变更审查

### statusMap 修复（4.2B）

**Before（用单字母/Emoji 匹配 XML 值）：**
```ts
modified: ['M'], added: ['A', '+'],
conflicted: ['C', '!'], deleted: ['D', '🗑'],
```

**After（匹配 XML 属性全称）：**
```ts
modified: ['modified'], added: ['added'],
conflicted: ['conflicted'], deleted: ['deleted'],
unversioned: ['unversioned'],
```

✅ 与 `FileItem.status`（`svn status --xml` 的 `@item` 属性值）一致。

### locale 联动新增（4.3）

**App.vue 初始化：**
```ts
onMounted(() => {
  const settingsStore = useSettingsStore()
  const { locale } = useI18n()
  settingsStore.load().then(() => {
    if (settingsStore.language && settingsStore.language !== 'system') {
      locale.value = settingsStore.language
    }
  })
})
```

**SettingsPage 保存后：**
```ts
async function handleSave() {
  await settingsStore.save()
  if (settingsStore.language !== 'system') {
    locale.value = settingsStore.language
  }
}
```

✅ 覆盖"启动加载"和"保存后切换"两个场景。
