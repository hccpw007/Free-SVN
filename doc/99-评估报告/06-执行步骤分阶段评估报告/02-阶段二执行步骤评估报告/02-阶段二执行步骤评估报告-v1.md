## §1 编译错误（P1 — 必须修复）

审查发现 2 处导致代码无法编译的错误。

### 1.1 `run_svn_sync()` 中 `child` 所有权冲突（2.2）

| 位置 | 2.2-SVN执行核心.md，`run_svn_sync()` 函数 |
|------|------------------------------------------|

```rust
let mut child = cmd.spawn()...?;

// 此处 child 被闭包捕获（move）→ 所有权转移
if let Ok(mut guard) = CURRENT_CHILD.lock() {
    *guard = Some(child.try_wait().ok().flatten()
        .map(|_| child)         // ← child 被 move 进闭包
        .unwrap_or(child));     // ← child 已被 move，第二次使用 → 编译错误
    drop(guard);
}

// 此处再次使用 child → 编译错误
let output = child.wait_with_output().map_err(AppError::Io)?;
```

**原因**：`std::process::Child` 没有实现 `Clone`/`Copy`。`map(|_| child)` 将 `child` 所有权移入闭包后，无法在后续的 `unwrap_or(child)` 和 `child.wait_with_output()` 中再次使用。

**修复建议**：同步函数中不需要将 `Child` 存入 `CURRENT_CHILD`——执行期间无法同时等待和持有所有权等待取消。改为不保存 child 句柄：

```rust
fn run_svn_sync(svn_path: &PathBuf, args: &[String], cwd: &str) -> Result<String, AppError> {
    let start = Instant::now();
    log::info!("svn {} (cwd: {})", args.join(" "), cwd);

    let mut cmd = Command::new(svn_path);
    cmd.args(args).current_dir(cwd).stdout(Stdio::piped()).stderr(Stdio::piped());
    let mut child = cmd.spawn()...?;

    // 同步执行：不保存 child 到 CURRENT_CHILD（无法同时等待和持有所有权）
    // 异步取消通过 CANCELLED 标志 + spawn_blocking 超时处理
    let output = child.wait_with_output().map_err(AppError::Io)?;
    // ... 后续处理
}
```

**影响**：`cargo build` 编译失败，2 处 `use of moved value: child` 错误。

### 1.2 `get_svn_path()` 使用 Tauri 1 API（2.2）

| 位置 | 2.2-SVN执行核心.md，`get_svn_path()` release 分支 |
|------|--------------------------------------------------|

```rust
#[cfg(not(debug_assertions))]
{
    let resource_dir = tauri::api::path::resource_dir()  // ← Tauri 1 API，Tauri 2 不存在
        .unwrap_or_else(|| PathBuf::from("."));
    // ... 平台路径
}
```

**原因**：Tauri 2 移除了 `tauri::api` 模块，`tauri::api::path::resource_dir()` 不存在。Release 构建时此分支被编译导致 `error[E0432]: unresolved import`。

**修复建议**：阶段二开发期间统一返回系统 `svn`，内置路径配置推迟到阶段七打包处理：

```rust
pub fn get_svn_path() -> PathBuf {
    // 开发阶段统一使用系统 svn
    // 阶段七打包时改为内置路径（通过全局 OnceCell 注入）
    PathBuf::from("svn")
}
```

**影响**：`cargo build --release` 编译失败。Debug 模式不受影响。
