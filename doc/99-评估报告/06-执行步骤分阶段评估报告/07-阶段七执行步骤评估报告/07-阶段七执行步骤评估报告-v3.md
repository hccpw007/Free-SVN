#### P1 — `unregister` 函数未定义（新增）

第 51 行 `return { register, unregister }` 引用了名为 `unregister` 的变量，但**该函数未在任何位置定义**。阅读第 32-52 行代码块：

```ts
export function useKeyboardShortcuts() {
  const router = useRouter()
  const fileListStore = useFileListStore()

  function register(s: Shortcut[]) { allShortcuts.push(...s) }
  // ← 此处应插入 function unregister(id: string) { ... } 但完全缺失

  function handler(e: KeyboardEvent) { ... }
  // ...
  return { register, unregister }  // TypeScript 错误：unregister 未定义
}
```

这会导致 TypeScript 编译错误 `Cannot find name 'unregister'`，严重性 P1。

**原因：** v1 修复时对 `Shortcut` 接口和 `unregister` 函数体的编辑被 GateGuard 拦截未生效，仅最后一行的 `return { register }` → `return { register, unregister }` 被成功更新。

**建议：** 在第 36 行 `function register(...)` 之后补充 `unregister` 函数实现：

```ts
function register(s: Shortcut[]) { allShortcuts.push(...s) }

function unregister(id: string) {
  for (let i = allShortcuts.length - 1; i >= 0; i--) {
    if (allShortcuts[i].id === id) allShortcuts.splice(i, 1)
  }
}
```

#### P1 — Shortcut interface 缺少 `id` 字段（新增）

第 28 行 `Shortcut` 接口定义：

```ts
interface Shortcut { keys: string; handler: ShortcutHandler; scope: 'dialog' | 'panel' | 'global' }
```

`id` 字段缺失。第 68 行的注册规则要求"所有 register 调用时传入带 `id` 的 Shortcut 对象"，且上述 `unregister(id)` 需通过 `allShortcuts[i].id` 查找。无 `id` 字段时 TypeScript 报错 `Property 'id' does not exist on type 'Shortcut'`。

**建议：** 接口追加 `id?: string`：

```ts
interface Shortcut { id?: string; keys: string; handler: ShortcutHandler; scope: 'dialog' | 'panel' | 'global' }
```

#### P1 — macOS register() 代码仍写入空 document.wflow（新增）

第 77-78 行实际执行代码：

```rust
fs::write(bundle.join("Contents/document.wflow"),
    "<?xml version=\"1.0\" encoding=\"UTF-8\"?>\n<plist version=\"1.0\"><dict></dict></plist>")?;
```

写入的是**空 XML**，不含任何 Automator action 定义。尽管后方的 NOTE 注释（第 108-164 行）包含完整的 XML 结构，但 AI 编码时**按代码执行而非按注释执行**，会直接复制第 77-78 行的字符串常量，生成的 `.workflow` bundle 在 Finder 中不可用。

**建议：** 将 document.wflow 写入内容替换为包含 `Run Shell Script` action 的 XML。最小可用示例：

```rust
let wflow = format!(
    r#"<?xml version="1.0" encoding="UTF-8"?>
<plist version="1.0"><dict>
<key>AMWorkflowActions</key><array><dict>
<key>action</key><key>AMAction</key><dict>
<key>AMActionName</key><string>Run Shell Script</string>
<key>AMAccepts</key><dict><key>Container</key><string>List</string>
<key>Optional</key><false/>
<key>Types</key><array><string>public.file-url</string></array></dict>
<key>AMParameterProperties</key><dict>
<key>COMMAND_STRING</key><string>{app} --svn-cmd {cmd} "$@"</string>
<key>inputMethod</key><integer>1</integer>
<key>shell</key><integer>0</integer>
</dict></dict></dict></array>
<key>AMWorkflowMetadata</key><dict>
<key>AMWorkflowSummary</key><string>SVN {name}</string>
</dict></dict></plist>"#,
    app = "/Applications/Free-SVN.app/Contents/MacOS/free-svn",
    cmd = cmd,
    name = name,
);
fs::write(bundle.join("Contents/document.wflow"), &wflow)?;
```
